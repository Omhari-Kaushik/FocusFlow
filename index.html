How to activate the real tab blocker:

  1. Navigate to chrome://extensions in your Google Chrome browser.
  2. Enable Developer Mode using the toggle switch in the top-right corner of the extensions page.
  3. Click the Load Unpacked button in the top-left corner.
  4. Select the focusflow-extension folder located inside the project's root directory.
  5. The extension will load and automatically block websites on your blacklist whenever the Distraction Shield is ON!
